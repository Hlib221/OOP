Тема: Програмування з використанням ООП

Мета роботи: Навчитись працювати з Класами та його основними конструкціями

Виконання роботи

В ході лабораторної роботи було розроблено програму, яка демонструє ключові принципи ООП:

1.  **Базовий клас та об'єкти:**
    * Розроблено клас `Vehicle` для опису транспортних засобів.
    * Реалізовано методи управління станом двигуна (`start`, `stop`) та отримання інформації.

2.  **Наслідування (Inheritance):**
    * Створено дочірні класи `Car` (автомобіль) та `Motorcycle` (мотоцикл).
    * Для автомобіля додано логіку витрати палива та розрахунку дистанції.
    * Для мотоцикла реалізовано унікальний метод виконання трюку.

3.  **Інкапсуляція (Encapsulation):**
    * Реалізовано клас `BankAccount` (Банківський рахунок).
    * Забезпечено захист балансу та PIN-коду через приватні атрибути.
    * Доступ до фінансових операцій можливий лише після перевірки пароля.

4.  **Поліморфізм (Polymorphism):**
    * Створено систему класів тварин (`Dog`, `Cat`, `Bird`), які мають спільний інтерфейс, але різну реалізацію методів.
    * Продемонстровано обробку списку різних об'єктів через єдину функцію.

5.  **Класові методи та змінні:**
    * У класі `Student` реалізовано лічильник створених об'єктів та метод для підрахунку середнього балу.
